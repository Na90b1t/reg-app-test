# Simple Auth App

Приложение с полноценной системой авторизации и регистрации на Angular с Node.js бэкендом.

## Быстрый старт

### 1. Установите зависимости фронтенда (из корня проекта):
```bash
npm install
```

### 2. Установите зависимости бэкенда:
```bash
cd server
npm install
cd ..
```

### 3. Запустите бэкенд (в первом терминале):
```bash
cd server
npm start
```
Сервер будет доступен на http://localhost:3000

### 4. Запустите фронтенд (во втором терминале, из корня проекта):
```bash
npm start
```
Приложение будет доступно на http://localhost:4200

### 5. Откройте в браузере:
- **Фронтенд:** http://localhost:4200
- **API Health Check:** http://localhost:3000/api/health

**⚠️ Важно:** Оба сервера должны работать одновременно!

## Особенности

- ✅ Регистрация пользователей с хешированием паролей
- ✅ Авторизация с JWT токенами
- ✅ Безопасное хранение паролей (bcrypt)
- ✅ Автоматическое сохранение сессии
- ✅ Защищенные API endpoints
- ✅ Поддержка типов пользователей (`user`, `mop`) и кнопка «Вход через Auth0» для агентов
- ✅ Простой и понятный код для обучения

## Технологии

### Фронтенд
- Angular CLI 16.2.16
- TypeScript
- RxJS

### Бэкенд
- Node.js + Express
- JWT для авторизации
- bcrypt для хеширования паролей
- JSON файл для хранения данных (можно заменить на БД)
